    Class Decoder

    Used to decode an Uint8Array into various kind of data.

    Example

    Decodes a binary string:

    const str = Decoder.decode(new Uint8Array([3, 0, 97, 98, 99]), (decoder: Decoder) => {
      const length: number = decoder.int16LE(); // decodes the string's length
      return decoder.string(length, { encoding: 'binary' }); // decodes the string as binary
    });

    // str => 'abc'

    string

    • string(length: number, __namedParameters?: DecoderStringOptions): string

      Reads a string from this Decoder.

      The string may be encoded in different formats:

      • utf-8 (default): encodes the string in utf-8 format.
      • binary: encodes the string in binary format.
      • hex: encodes the string in hex format (ex: 0AF3).
      • base64: encodes the string in base64 format (ex: YWJjZA==).
      • any valid labels: encodes the string according to this format, if it is supported.
