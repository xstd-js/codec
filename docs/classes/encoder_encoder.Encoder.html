    Class Encoder

    Used to encode various kind of data into an Uint8Array.

    Example

    Encodes a binary string:

    const str: string = 'abc';

    const bytes = new Encoder()
     .int16LE(str.length) // encodes the string's length
     .string(str, { encoding: 'binary' }) // encodes the string as binary
     .toUint8Array();

     // bytes => [3, 0, 97, 98, 99]

    string

    • string(value: string, __namedParameters?: EncoderStringOptions): this

      Writes a string in this Encoder.

      The string may be encoded in different formats:

      • utf-8 (default): encodes the string in utf-8 format, and appends the bytes into this Encoder's bytes.
      • binary: assumes the string is a binary string, and appends each byte of this string, into this Encoder's bytes.
      • hex: assumes the string is an hex string (ex: 0AF3), and appends each decoded byte of this string, into this Encoder's bytes.
      • base64: assumes the string is an base64 string (ex: YWJjZA==), and appends each decoded byte of this string, into this Encoder's bytes.
